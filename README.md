# 📚 Sistema de Gestión de Biblioteca

Aplicación web desarrollada en Django para gestionar préstamos de libros y documentos. Permite administrar usuarios, catálogo y el control de préstamos y devoluciones.

## 👨‍💻 Mi contribución

- Desarrollo completo del frontend  
- Implementación de interfaces con HTML, CSS y JavaScript  
- Integración del frontend con el backend (Django)  
- Mejora de la visualización y experiencia de usuario  

## 🛠️ Tecnologías utilizadas

- Python (Django)  
- HTML, CSS, JavaScript  
- SQL (Base de datos)  

## 📦 Dependencias

El proyecto utiliza las siguientes librerías principales:

- Django  
- Django REST Framework  
- Simple JWT (autenticación)  
- PostgreSQL (psycopg2)  
- Gunicorn  
- Whitenoise  

Para instalar todas las dependencias:

pip install -r requirements.txt

## ▶️ Cómo ejecutar el proyecto

### Backend (Django)

cd backend  
python -m venv .venv  
.venv\Scripts\activate  
pip install -r requirements.txt  
python manage.py migrate  
python manage.py runserver  

### Frontend

cd frontend/sistema-biblioteca  
npm install  
npm run dev
